# AllStocks
A csv file containing symbol, Yahoo Finance ticker suffix, exchange, exchange name. 
Yahoo Finance ticker can be made by simply combining the symbol and yfinance suffix. 
Example - Symbol: INFY, YFinance suffix: .NS, Yahoo Finance Ticker - INFY.NS


Exchanges included - NSE, BSE, NASDAQ, NYSE, KOSPI, SIX, TSE, LSE, DUS, FWB. 
Added currenncies.csv with thousands of forex pairs.
